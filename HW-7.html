Suppose the target distribution is \(g_X(s) = s^{-3 / 2} \mathbb{I}(s \geq 1)\). A slice sampler is used that works as follows given the current state \(X\).

  1. Draw \(Y\) uniformly from 0 to \(1 / X^{3 / 2}\).

  2. Draw \(X\) uniformly from 1 to \(1 / Y^{2 / 3}\).

Implement this single step in R, then use it to take 1000 burn in steps followed by 1000 states of this chain, and use it to estimate \(\mathbb{P}(X \geq 2)\). Replicate the chain run 10 times and report your result as \(a \pm b\).

Step function

step_splice_p1<-function(x){
  y<-runif(1)*1/x^(3/2)
  x<-runif(1, min=1, max=1/y^(2/3))
  return(x)
}

Mchain

set.seed(1234567)
slice_sampler_p1<-function(steps){
  burnin<-steps
  datasteps<-steps
  x<-1
  for(i in 1:burnin){
    x<-step_splice_p1(x)
  }
  res<-rep(0, datasteps)
  for(i in 1:datasteps){
    res[i]<-(x>=2)
    x<-step_splice_p1(x)
  }
  return(res)
}

This is the step function

step_prod_slice_p3<-function(x, u){
  y1<-u[1]*sin(x)
  y2<-u[2]*cos(x)
  y3<-u[3]*cos(x)
  #lower bound
  lb<-asin(y1)
  ub<-min(acos(y2), acos(y3))
  x<-runif(1, min=lb, max=ub)
  return(x)
}

Below we run the Markov Chain. Using wolfram alpha, I found that the normalizing constant \(C\) for \(g_X(x)\) is \(3\), so I compared the output of my Markov Chain to \(3 \cdot g_X(x)\).

Suppose that \(X \sim \textsf{Unif}([0, 5])\), and \(M \sim \textsf{Unif}([-1, 1])\). Find analytically the probability that \(X + M \in [0, 5]\).

Let’s first focus on the cases where \(X+M\) is guaranteed to lie in \([0,5]\). If \(X \in [1, 4]\), then \(X+M\) is guaranteed to lie in \([0,5]\). The probability that \(X \in [1,4]=\frac{m([1,4])}{m([0,5])}=\frac{3}{5}\). Now consider the case where \(X \in [0,1)\). The expected value of \(X\) in this region is \(\mathbb{E}[X]=1/2\). For \(X=1/2\), \(X+M \in [0,5]\) with probability \(3/4\). Thus, \(3/4\) is the expected probability of remaining in \([0,5]\) if \(X \in [0,1)\). The probability that \(X \in [0,1)\) in the first place is \(1/5\). The joint probability, of \(X \in [0,1)\) AND \(X+M \in [0,5]\) is \(3/4 \cdot 1/5 = 3/20\). The same exact logic holds for \(X \in (4,5]\). The expected probability of remaining in \([0,5]\) if \(X \in (4,5]\) is \(3/4\), and the probability of \(X \in (4,5]\) in the first place is \(1/5\). So the joint probability is \(3/20\) as well. So the total probability of \(X+M \in [0,5]\) is the sum of these probabilities: \(3/5 + 3/20 + 3/20 = \boxed{9/10}\).

step_p6<-function(x, u){
  xnew<-x+(2*u-1)
  if(xnew<=5 & xnew>=0){
    return(xnew)
  }
  else{
    return(x)
  }
}
set.seed(1234567)

p6_mchain<-function(steps){
  burnin<-steps
  datasteps<-steps
  m1<-runif(burnin)
  x<-3
  for(i in 1:burnin){
    x<-step_p6(x, m1[i])
  }
  res<-rep(0, datasteps)
  m2<-runif(datasteps)
  for(i in 1:datasteps){
    xnew<-step_p6(x, m2[i])
    #return if new state equals old state
    #i.e. if move was rejected b/c it would land outside state space
    res[i]<-(x!=xnew)
    x<-xnew
  }
  return(res)
}

res<-replicate(10, mean(p6_mchain(steps=1000)))

  mean=mean(res),
  se=sd(res)/sqrt(10)
) %>% kable()
